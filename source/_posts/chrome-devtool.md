**常用快捷键一览**

* DevTool面板

|  快捷键   |  功能  |
| -------- | ------ |
|`ctrl + +/-` | 窗口放大/缩小 |
|`ctrl + 0` | 窗口默认大小 |
| `ctrl + shift + j` | 开启Dev模式，并定位到console页签 |
| `ctrl + [` | 页签切换(向左) |
|`ctrl + shift + p`|DevTools Command Menu|

* Source页签

|  快捷键   |  功能  |
| -------- | ------ |
|`ctrl + g` | 跳转行 |
|`ctrl + p` | 打开文件 |
|`F11`|进入函数|
|`shift + F11`、`ctrl + shift + ;`|跳出函数|

* Console页签

<kbd>dddd</kbd>

|  快捷键   |  功能  |
| -------- | ------ |
|<code>ctrl + &#96;</code> | 聚焦到console |
